== Description ==

It's a very simple plugin for protect your site content. "WP Protect Content" plugin have an option for disable to copy of content on right click of mouse and also have an option for disable right click.

Note : please don't forget to leave your valuable feedback and suggestions for make this plugin much more better compare to other plugins.

Send your query on raghunath.0087@gmail.com


= `Features` = 
 * An option for Disallow copy of the content
 * An option for Disallow right click 
 
== Frequently Asked Questions ==

1.) Can i disallow copy of the content?

 * Yes, you need to checked corresponding check box from plugin settigs page.
 
2.) Can i disallow right click?

 * Yes,you need to checked corresponding check box from plugin settigs page.

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The settings page will be available under settings menu (Settings >> WP Version Remover)
